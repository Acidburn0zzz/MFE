Tools

Five tools, with highly different designs and implementations, are available from the latest MFE:

  • the Theorem Prover (ITP)
  • the Sufficient Completeness Checker (SCC)
  • the Church-Rosser Checker (CRC)
  • the Coherence Checker (ChC)
  • the Maude Termination Tool (MTT)

Originally, these tools where independent tools. The interaction with each of them remains as in its stand-alone version after selecting it in the MFE environment (with a select tool < tool > . commnand).
